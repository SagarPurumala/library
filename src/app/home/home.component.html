Library Management System

Library management system is all about organizing, managing the library and library-oriented tasks. It also involves maintaining the database of entering new books and the record of books that have been retrieved or issued, with their respective dates. Library Management System is a computerized system which helpsuser(librarian) to manage the library daily activity in electronic format. It reducesthe risk of paper work such as file lost, file damaged and time consuming.It can help user to manage the transaction or record more effectively and time-saving
